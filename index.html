What I've Built

NLP / RAG

Moksh AI v3.0 — Hybrid Agentic RAG System

Agentic AI assistant that routes queries between FAISS vector memory, DuckDuckGo web search, or hybrid mode using temporal keyword detection and L2 confidence thresholds. Dual-LLM with Llama 3 + Gemini 2.0 Flash fallback. 92% on 25-question benchmark. Research published on Zenodo.

PythonLlama 3 FAISSall-MiniLM Streamlit
Computer Vision

Deepfake Face Detector

ResNet-50 binary classifier trained on 140,000 face images — 94.05% test accuracy and ROC-AUC 0.98. Mixed precision on RTX 4060. Streamlit interface for real-time image and video detection with under 12ms latency per frame.

PyTorchResNet-50OpenCV
Backend

Customer Churn Prediction API

ML pipeline on IBM Telco dataset — ROC-AUC 0.836, 80% accuracy. Production Flask REST API with batch scoring, data drift detection, and date-stamped model versioning for monthly retraining.

FlaskScikit-learnPandas
Computer Vision

FashionNet

3-block CNN from scratch in PyTorch for apparel classification. Pushed accuracy from 88.2% to 93.15% using BatchNorm, Cosine LR, dropout, and label smoothing over 20 epochs. Precision, Recall, F1 all at 0.93 across 10 classes.

PyTorchCNNPython

Event Management System

Full-stack college event registry with admin panel, user dashboards, and CRUD operations for event registration. PHP backend with MySQL database.
